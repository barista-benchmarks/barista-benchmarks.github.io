Barista
Barista is a modern, open, and diversified benchmark suite for the JVM, aimed at testing JIT and AOT compilers, garbage collectors, profilers, analyzers and other tools.

Barista is a benchmarking suite that contains a range of microservice applications implemented using various popular JVM microservice frameworks.

The benchmark suite is developed with the goal of enabling the JVM developer community to make improvements and optimizations with the performance of microservices in mind.

Barista provides key metrics for evaluating microservices, such as throughput, latency, startup time, memory usage, and more!

The Barista harness facilitates microservice developers to easily benchmark their apps and determine the optimal deployment configuration for their use case.

Barista is completely open source, with the code available at GitHub, and is designed as part of an ongoing, collaborative effort to determine a set of modern JVM microservice benchmarks.
